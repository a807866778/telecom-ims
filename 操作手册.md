## 系统登录

### 登录流程

1. 访问系统地址
2. 输入用户名和密码
3. 点击"登录"按钮

### 默认账号

| 角色 | 用户名 | 密码 | 权限 |
|------|--------|------|------|
| 超级管理员 | admin | admin123 | 全部功能 |

### 安全提示

- 首次登录后请立即修改密码
- 密码建议包含：大小写字母、数字、特殊字符
- 定期更换密码

## 首页仪表盘

### 功能说明

首页展示系统核心数据概览。

### 统计卡片

- **项目总数**: 当前系统中所有项目的数量
- **物料总数**: 商品库中的物料数量
- **今日入库**: 今日入库记录的条数
- **今日出库**: 今日出库记录的条数

### 利润趋势图

展示经营数据的利润走势：

- **时间维度**: 今日 / 本月 / 本年
- **数据指标**: 收入、成本、利润
- **图表类型**: 柱状图

### 操作入口

- 点击模块卡片可快速进入对应功能页面
- 点击"全部"可查看完整列表

## 采购管理

### 采购订单

**功能**: 创建和管理采购订单

**操作步骤**:
1. 进入"采购管理" > "采购订单"
2. 点击"新增采购订单"
3. 填写订单信息（供应商、采购明细等）
4. 点击"保存"

**字段说明**:
| 字段 | 说明 | 必填 |
|------|------|------|
| 订单编号 | 自动生成 | 是 |
| 供应商 | 选择供应商 | 是 |
| 预期到货日期 | 预计收货日期 | 否 |
| 备注 | 补充说明 | 否 |

### 采购收货

**功能**: 记录采购到货情况

**操作步骤**:
1. 进入"采购管理" > "采购收货记录"
2. 点击"新增收货记录"
3. 关联采购订单或手动录入
4. 填写收货明细（物料、数量、单价等）
5. 可上传收货照片
6. 点击"保存"

### 采购退货

**功能**: 处理退货业务

**操作步骤**:
1. 进入"采购管理" > "采购退货"
2. 点击"新增退货"
3. 选择关联的收货记录
4. 填写退货原因和明细
5. 点击"保存"

## 出库管理

### 出库单

**功能**: 记录项目出库

**操作步骤**:

4. 填写出库明细（物料、数量、单价等）

**功能**: 处理现场多领料退库

**操作步骤**:
1. 进入"出库管理" > "退库单"
2. 点击"新增退库"
3. 选择关联项目
4. 填写退库明细
5. 点击"保存"

## 项目管理

**功能**: 管理所有项目

## 仓储管理

### 库存查询

**功能**: 查看当前库存

**操作步骤**:
1. 进入"仓储管理" > "库存查询"
2. 可按物料名称、编号筛选
3. 查看库存数量、预警状态

**库存预警**: 
- 当库存低于最低库存时会显示预警

### 入库记录

**功能**: 查看入库历史

**操作步骤**:
1. 进入"仓储管理" > "入库记录"
2. 可按日期、供应商筛选
3. 点击记录可查看详情和照片

### 出库记录

**功能**: 查看出库历史

**操作步骤**:
1. 进入"仓储管理" > "出库记录"
2. 可按日期、项目筛选
3. 点击记录可查看详情

### 报溢报损

**功能**: 处理库存盘盈盘亏

**操作步骤**:
1. 进入"仓储管理" > "报溢报损"
2. 点击"新增调整"
3. 选择物料、调整类型（报溢/报损）
4. 填写数量和原因
5. 点击"保存"

### 流向查询

**功能**: 追踪物料流向

**功能**: 入库时间节点 → 出库时间节点

### 库存盘点

**功能**: 月度库存盘点

**操作步骤**:
1. 进入"仓储管理" > "库存盘点"
2. 选择盘点月份
3. 系统生成盘点单
4. 填入实际库存数量
5. 确认盘点结果

5. 点击"保存"

**字段说明**:
| 字段 | 说明 |
|------|------|
| 商品名称 | 商品的正式名称 |

| 规格/型号 | 商品规格信息 |
| 进货价 | 采购价格 |
| 销售价 | 销售价格 |
| 最低库存 | 库存预警阈值 |

5. 点击"保存"

### 项目合同

## 常见问题

### 忘记密码怎么办？

联系系统管理员重置密码。

### 如何上传文件？

在各模块的表单中，找到"上传附件"或"上传照片"按钮，点击后选择文件即可。

### 库存为负数怎么办？

检查是否有未审核的出库单，或执行报溢操作补充库存。

### 如何查看历史数据？

在各列表页面，可通过日期筛选器选择时间范围进行查询。
