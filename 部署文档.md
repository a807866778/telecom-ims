# 电信IMS系统 - 部署文档

## 系统概述

电信IMS系统是一套基于 Cloudflare Workers + Next.js 15 的企业级电信设备进销存管理系统。

### 技术栈

- **前端框架**: Next.js 15.3.0 + React 19
- **样式**: Tailwind CSS v4
- **数据库**: Cloudflare D1 (SQLite)
- **对象存储**: Cloudflare R2
- **部署平台**: Cloudflare Workers
- **构建工具**: OpenNext Cloudflare

### 系统模块

1. **首页仪表盘** - 数据统计、利润图表、待办事项
2. **采购管理** - 采购订单、收货记录、退货
3. **出库管理** - 出库单、退库单
4. **项目管理** - 项目列表、项目汇总
5. **仓储管理** - 库存查询、入库记录、出库记录、报溢报损、库存盘点
6. **档案管理** - 商品管理、经销商档案、项目合同
7. **人员管理** - 人员档案、健康档案、证照档案
8. **安规培训** - 培训内容、培训考核、培训记录
9. **售后管理** - 客户投诉、设备售后
10. **设置** - 账号管理、角色管理、通知管理

## 部署要求

### 环境要求

- Node.js 20.x+
- npm 10.x+
- Wrangler CLI 3.x+
- Cloudflare 账号

### Cloudflare 资源准备

1. **创建 D1 数据库**
```bash
wrangler d1 create telecom-ims
```

2. **创建 R2 存储桶**
```bash
wrangler r2 bucket create telecom-ims-media
```

3. **记录资源 ID**
   - D1 Database ID
   - R2 Bucket 名称

## 部署步骤

### 1. 克隆代码

```bash
git clone https://github.com/a807866778/telecom-ims.git
cd telecom-ims
```

### 2. 安装依赖

```bash
npm install
```

### 3. 配置环境

更新 `wrangler.toml` 中的配置：

```toml
name = "telecom-ims"
main = ".open-next/worker.js"

[[d1_databases]]
binding = "DB"
database_name = "telecom-ims"
database_id = "你的-D1-Database-ID"

[[r2_buckets]]
binding = "MEDIA_BUCKET"
bucket_name = "telecom-ims-media"

[vars]
NEXT_PUBLIC_APP_URL = "https://telecom-ims.你的workers子域名.workers.dev"
```

### 4. 初始化数据库

```bash
# 执行初始化脚本
npx wrangler d1 execute telecom-ims --file="drizzle/init.sql" --remote
npx wrangler d1 execute telecom-ims --file="drizzle/0000_rich_squadron_supreme.sql" --remote
# ... 执行其他迁移文件
```

或使用 Drizzle Kit:

```bash
npx drizzle-kit push
```

### 5. 构建项目

```bash
npm run build:cf
```

### 6. 部署到 Cloudflare

```bash
npx wrangler deploy
```

### 7. 配置域名（可选）

在 Cloudflare Dashboard 中：
1. 进入 Workers & Pages
2. 选择 telecom-ims
3. 设置自定义域名

## 运维指南


```bash

```

### 常见问题

#### 构建失败 (Windows EIO 错误)

**问题**: Windows 长路径导致 `fs.cpSync` 报错

**解决**: 移动项目到短路径，如 `C:\projects\telecom-ims`

#### @libsql/client 解析失败

**问题**: esbuild 无法解析 `@libsql/client`

**解决**: 修改 `node_modules/@opennextjs/cloudflare/dist/cli/build/bundle-server.js`，在 external 数组中添加：
```javascript
"@libsql/client",
"@libsql/core",
"@libsql/hrana-client",
"@libsql/isomorphic-fetch",
"@libsql/isomorphic-ws",
"drizzle-orm",
```
